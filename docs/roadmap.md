# AgentWire Roadmap

## v1.0 (MVP) - Foundation & Observability
- [ ] Core ingestion API (OpenTelemetry compatible)
- [ ] PostgreSQL + ClickHouse storage implementation
- [ ] Next.js Real-time Dashboard (NOC view)
- [ ] Basic AI Packet Inspector
- [ ] Token Usage and Cost Analytics
- [ ] .NET and Python SDKs
- [ ] Docker Compose for easy self-hosting

## v2.0 - Security & Interactivity
- [ ] Real-time Security Scanner (Prompt Injection, PII leakage)
- [ ] Replay Engine (Re-run executions from the UI)
- [ ] Prompt Version Control & Diff Viewer
- [ ] Alert Engine (Webhooks, Slack, Email integrations)
- [ ] Interactive Execution Graph (React Flow)
- [ ] Advanced Search Engine
- [ ] Kubernetes Helm Charts

## Enterprise Edition - Governance & Scale
- [ ] Multi-tenant SaaS architecture
- [ ] Role-Based Access Control (RBAC) & SSO (SAML/OIDC)
- [ ] Custom Security Rules Engine
- [ ] Data Retention Policies and Cold Storage Tiering (S3)
- [ ] AI-powered Failure Analysis and Optimization Suggestions
- [ ] Advanced Cost Intelligence (Predictive analytics)
- [ ] Audit Logs
